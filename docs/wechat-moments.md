## 文案 B — 技术展示风

> 周末项目上线了！AutoCoin — 一个开源的 AI 记账工具 🤖
>
> 亮点功能：
> ✅ 拍照/截图 → AI 识别 → 自动入账（支持 GPT-4o、Gemini、通义千问、智谱、DeepSeek，自动 Fallback）
> ✅ 支付宝 CSV + 微信 XLSX 一键导入，自动去重
> ✅ 收支概览、月度趋势、分类饼图，数据可视化全覆盖
> ✅ 自部署，JWT 多用户，SQLite 轻量运行
> ✅ FastAPI + 原生 JS SPA，零框架依赖
>
> 在线试用 → https://autocoin.zhiqin.org
> GitHub → https://github.com/hnkjdaxzzq/autocoin
>
> 欢迎 Star ⭐ 和反馈！
