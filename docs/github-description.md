## About 区域长描述

🪙 AutoCoin — 一款基于 Web 的个人记账工具。支持支付宝/微信账单文件导入，以及通过多模态大语言模型（LLM）智能识别收据照片/截图并自动入账。提供多维度收支统计、图表可视化，支持多用户隔离与移动端适配。

**核心亮点：**
- 📷 拍照/截图即可记账 — 支持 OpenAI、Gemini、智谱、通义千问、DeepSeek 等多家大模型，自动 Fallback
- 📂 一键导入支付宝 CSV & 微信支付 XLSX，自动去重
- 📊 收支概览、月度趋势、分类占比、每日明细，丰富图表
- 🔒 自部署、数据私有，JWT 多用户隔离
- 📱 移动端完整适配，支持手机拍照直接识别

🔗 在线体验：https://autocoin.zhiqin.org
